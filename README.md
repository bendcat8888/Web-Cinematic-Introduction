## 🎯 Project Overview
The **InnoGen Pharmaceuticals Cinematic Web Experience** is a static corporate website designed to present InnoGen's brand, pharmaceutical focus, and healthcare ecosystem through a polished, modern interface.

It emphasizes brand clarity, accessibility of information, responsive presentation, and a premium visual direction aligned with InnoGen Pharmaceuticals Inc.'s message: **We Value Life**.

### 🌟 Core Website Sections
1. **Corporate Identity:** Highlights InnoGen Pharmaceuticals Inc., its brand positioning, and company message.
2. **Research & Laboratories:** Presents science-forward content, laboratory visuals, and innovation-focused page sections.
3. **Doctors Portal:** Provides a dedicated entry point for healthcare professionals.
4. **Health & News Content:** Organizes health tips, news, events, and insights into standalone pages.
5. **Career & Partnership Pathways:** Includes a working-with-us page for professional engagement.

## ⚙️ Development & Quick Start

### Prerequisites
- A modern web browser
- Git, if deploying to GitHub
- Optional: a local static server for previewing files

### Local Preview
Open `index.html` directly in your browser, or serve the folder locally:

```bash
python -m http.server 8765
```

Then visit:

```text
http://127.0.0.1:8765/
```

## 🌐 GitHub Pages Deployment

### Push to GitHub
```bash
git add .
git commit -m "Add project README"
git push
```

### Enable GitHub Pages
1. Open the repository on GitHub.
2. Go to **Settings** → **Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Select the `main` branch and `/root` folder.
5. Save the settings.

The website will be available at:

```text
https://bendcat8888.github.io/<repository-name>/
```
